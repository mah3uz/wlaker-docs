Select screens/windows via xdg-desktop-portal.

## Configure

```toml
[builtins.xdph_picker]
name = "xdphpicker"
placeholder = "Screen/Window Picker"
switcher_only = true
show_sub_when_single = true
hidden = true
```

Enable when you want to launch screen/window picking from Walker.
