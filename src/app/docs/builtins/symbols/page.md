Common symbols and glyphs. Copy to clipboard with Enter.
---
## Configure

```toml
[builtins.symbols]
name = "symbols"
placeholder = "Symbols"
switcher_only = true
history = true
typeahead = true
exec = "wl-copy '%RESULT%'"
```

Great for quickly grabbing checkmarks, arrows, and math symbols.
